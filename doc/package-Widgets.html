Package Widgets

Classes summary

BattleMessageWidget Displays a BattleMessage
CharactermodifierWidget Displays (or more often: returns) the Charactermodifier part of an element's popup content, e.g. for Items and Effects
CharacterProfileWidget Displays a little Character profile
CharacterSkillMenuWidget Displays a dropdown menu with active non-battle skills
CharacterStatsWidget Displays some important Character stats
EffectWidget Displays an Effect (button style)
EquipmentWidget Displays the current equipment of the Character
ItemGroupWidget Displays an Item group (like weapons, accessories, etc.)
ItemWidget Displays an Item adds additional links like "equip" or "unequip", depending on the context in which the item is to be displayed
QuestWidget Displays a Quest. Adds links to accept or reject the item, depending on its state
RequirementWidget Displays (or more often: returns) the Requirement part of an element's popup content, e.g. for Items
UserFlashesBasic Displays UserFlash messages, if there are any
UserMenuWidget Displays user-based actions in the main menu (login, logout, etc.)
