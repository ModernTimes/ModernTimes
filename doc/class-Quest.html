Class Quest

Provides some basic methods for handling quests. Has to be "overridden" by SpecialnessBehavior classes to do anything useful.

Adds a couple of standard event listeners to events triggered by Character. If you add more event handlers, don't forget to add a line to detachFromCharacter().

See BaseQuest for a list of attributes and related Models
