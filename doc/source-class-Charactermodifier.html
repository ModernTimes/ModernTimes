  5 /**
  6  * Data basis for Charactermodifier behavior for items, skills, etc.
  7  * Adds a couple of standard event listeners to stat calculation events in 
  8  * the Character model.
  9  *
 10  * Implemented so far:
 11  * - calcAttributes
 12  * - calc combat stats (critChance, resistances, bonus damage)
 13  * - dropCash/Favours/Kudos
 14  * - dropItem chance
 15  *
 16  * If you add more event handlers, don't forget to add a line to
 17  * detachFromCharacter()
 18  * 
 19  * Can be "overridden" by specialness behavior classes for items, skills, etc.
 20  * 
 21  * See BaseCharactermodifier for a list of attributes and related Models
 22  * 
 23  * @todo gainXp/Resoluteness/Willpower/Cunning Abs/Perc
 24  * 
 25  * @uses CharacterModifierBehavior
 26  * @uses SpecialnessBehavior
 27  * @package Character
 28  */
