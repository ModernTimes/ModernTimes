Class SpecialnessBehavior

Makes it possible to define special behavior for specific entries for certain models, like special monsters, items, areas, etc.

Important: If a model can have specially defined behaviors, do not call that model's function directly. Use the $model->call('functionName', params) syntax instead. SpecialnessBehavior will then redirect the call to either the standard model's function or to the special function of a class defined by the attribute specialClass in the model entry.
